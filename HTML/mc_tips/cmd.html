GAMEMODE

The "gamemode" commands in Minecraft are a set of powerful tools that allow players and operators to control their gameplay mode and creative abilities within the game. These commands are essential for changing the game's dynamics and mechanics, providing a range of creative and survival options to suit different playstyles and scenarios. There are four primary "gamemode" commands in Minecraft: /gamemode survival (or /gamemode 0): This command switches the player or target into Survival mode. In Survival mode, players must gather resources, survive hostile mobs, and manage their hunger and health, providing a challenging and immersive experience. /gamemode creative (or /gamemode 1): This command changes the player or target to Creative mode. Creative mode allows players to have unlimited resources and the ability to fly, making it ideal for building, designing, and experimenting without limitations. /gamemode adventure (or /gamemode 2): The Adventure mode is designed for custom maps and adventure scenarios. Players in Adventure mode cannot break blocks unless they have the proper tools. /gamemode spectator (or /gamemode 3): Spectator mode allows players to freely explore the game world as if they were a ghost.

EFFECT

There are two primary "effect" commands in Minecraft: /effect [player] [effect] [seconds] [amplifier] [hideParticles]: This command is used to apply a status effect to a player or entity. It includes several parameters: [player]: The target player or entity to apply the effect. [effect]: The specific status effect to apply (e.g., "minecraft:speed" for speed, "minecraft:strength" for strength, etc.). [seconds]: The duration of the effect in seconds. [amplifier]: The amplifier level of the effect (e.g., 0 for the base effect, 1 for the first level, etc.). [hideParticles]: An optional parameter to hide particles (true or false) associated with the effect. /effect clear [player] [effect]: This command is used to remove a specific effect from a player or entity. [player]: The target player or entity from which to remove the effect. [effect]: The specific effect to remove.

GAMERULE

Game Rules in Minecraft are a set of commands that allow players and server operators to modify various aspects of the game's behavior and mechanics. These rules provide a way to customize the gameplay experience, making the game more versatile and adaptable to different scenarios and preferences. Some common examples of game rules in Minecraft include: /gamerule doDaylightCycle: This command controls whether the day-night cycle in the game continues. Setting it to "false" makes the time stand still, while "true" allows the cycle to progress. /gamerule doMobSpawning: It toggles mob spawning on and off. When set to "false," hostile mobs won't spawn, providing a peaceful environment. /gamerule keepInventory: This rule determines whether players keep their inventory upon death. If set to "true," players retain their items; if set to "false," items are dropped upon death.
